常见问题

您可以通过以下步骤注册账户：

  1. 点击首页的"注册"按钮
  2. 填写用户名、邮箱和密码
  3. 阅读并同意用户协议
  4. 点击"注册"完成账户创建

如果您忘记了密码，可以通过以下方式重置：

  1. 在登录页面点击"忘记密码"
  2. 输入您的注册邮箱
  3. 查看邮箱中的密码重置邮件
  4. 点击邮件中的链接设置新密码

系统会自动保存您的对话记录：

  • 对话记录会自动保存在左侧历史记录栏中
  • 您可以为对话添加标题以便识别
  • 在设置中可以开启自动保存功能
  • 支持导出对话记录为文本文件

您可以通过以下步骤更改个人设置：

  1. 点击右侧边栏的"用户中心"
  2. 在用户中心页面修改昵称、语言等信息
  3. 点击"设置"可以配置系统偏好
  4. 所有更改会自动保存

系统支持以下快捷键操作：

  • Enter：发送消息
  • Shift + Enter：换行
  • Ctrl + N：新建对话
  • Ctrl + Shift + N：清空当前对话
  • Ctrl + ,：打开设置
